## git 명령어

```
git init // 최초 한번 - git 저장소 만들기

git status // 현재 폴더 상태 확인
git add . // 현재 폴더 수정 사항 모두 tracking
git commit -m '커밋메시지' // 등록된 tacking 파일 commit (git add로 등록된)
git push origin master // [origin] 이라는 서버에 [master] 라는 branch 로 로컬파일 전송

```
